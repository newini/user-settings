# User setting files

## Contents

* **bashrc** - for bash shell

* **vimrc**, **vim/after/syntax/cpp.vim** - for vim

* **latexmkrc** - for latexmk

## Usage
Type below
```
source setup.sh
```
It will add line into **.bashrc**, **.vimrc**, **.vim/after/syntax/cpp.vim**, 
and copy **latexrc** to **.latexmkrc**
